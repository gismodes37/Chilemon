# Support

If you need help with ChileMon, there are several ways to get support.

---

# GitHub Issues

The primary support channel is GitHub Issues.

You can:

- report bugs
- ask technical questions
- suggest improvements

Open an issue here:

https://github.com/gismodes37/Chilemon/issues

---

# Email Contact

For direct contact you can email the project maintainer.

Email:

ca2iig@qsl.net

Please include:

- your operating system
- ASL version
- node number
- description of the problem

---

# Before Requesting Support

Please verify:

- ChileMon was installed using the installer
- Apache is running
- PHP extensions are installed
- SQLite is available
- Asterisk is operational

---

# Tested Environments

ChileMon has been tested on:

- Raspberry Pi 3B+ with ASL3
- Debian 13 virtual machine (Proxmox)

Other environments may work but have not yet been fully validated.

---
